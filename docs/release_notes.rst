Release and Version History
===========================

0.3.0 (In Development)
----------------------

Features
^^^^^^^^

* Wrote lots of general documentation. See the ``docs`` directory.
* Wrote :doc:`writing_a_plugin`, a document outlining Drone Python
  plugin development.
* Added a :command:`create-drone-py-plugin` command for quickly
  bootstrapping a new plugin from a template that we maintain. See
  :doc:`writing_a_plugin` for more details.

0.2.0 (2015-12-08)
------------------

Features
^^^^^^^^

* You may now ``import drone`` and use ``drone.plugin.input`` directly
  without additional imports. (msteinert)

0.1.0 (2015-11-13)
------------------

* Initial release.
